Git

  • git init: Инициализирует новый репозиторий Git.
  • git add: Добавляет изменения в область индексации (staging area).
  • git commit: Сохраняет изменения из области индексации в локальный репозиторий.
  • git status: Показывает текущий статус репозитория.
  • git log: Отображает историю коммитов.
  • git branch: Создает, перечисляет или удаляет ветки.
  • git checkout: Переключается между ветками.
  • git merge: Объединяет изменения из одной ветки в другую.
  • git push: Отправляет коммиты в удаленный репозиторий.
  • git pull: Получает изменения из удаленного репозитория и объединяет их с локальной веткой.
